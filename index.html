Vellum

A drop-in overlay that makes any page's text editable. Click. Edit. Copy as agent-ready markdown.

Bookmarklet

Drag this to your bookmarks bar, then click it on any page:

javascript:(()=>{const s=document.createElement('script');s.src='https://cdn.jsdelivr.net/gh/wonderlandgardens/vellum@main/vellum.js';document.head.appendChild(s);})()

Script tag

<script src="https://cdn.jsdelivr.net/gh/wonderlandgardens/vellum@main/vellum.js"></script>

npm

npm i vellum
import { mount } from "vellum";
mount();

Try it

This paragraph is editable. Click any text on this page, edit it, then click Copy markdown in the bottom-right toolbar.

Headings, paragraphs, list items, and buttons are all armed by default. Edits are saved to localStorage; Reset restores originals in place.
